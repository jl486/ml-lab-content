It would be useful if we were able to refer to an outcome of an experiment. For example, we may want to mathematically represent a problem like the sum of two die rolls. To do this, we could let $X_1$ and $X_2$ be the outcomes of the dice ($1$, $2$, $3$, $4$, $5$, and $6$). The expression 

$$
X_1 + X_2
$$

would represent the sum of the two die rolls. Here, $X_1$ and $X_2$ are *random variables*. A **random variable** is a function whose output value is an outcome of a particular experiment.

## Types

Random variables can be either *discrete* or *continuous*.

* **Discrete random variables** have values that are countable or countably infinite. For example, The outcome of a die roll is a discrete random variable.
* **Continuous random variables** can take on any value in a continuous range, such as height or duration.

## Probability distributions

A **probability distribution** describes how the probabilities are spread out over the possible values of a random variable. This essentially defines the random variable.

In machine learning, probability distributions are important for visualizing and analyzing variables in a dataset.

### Discrete random variables

#### Probability mass function

For a discrete random variable $X$, its distribution is given by the **probability mass function (PMF)**, which takes in a value $x$ and tells you the probability of the random variable $X$ being equal to that value:

$$
p_X(x) = P(X = x)
$$

A PMF must satisfy these conditions:

* $p_X(x) > 0$ for all $x \in \Omega$
* $\sum_{x \in \Omega} p_X(x) = 1$

#### Cumulative distribution function

What if we wanted to find the probability of outcomes in an interval or below a certain threshold? We can find these probabilities with the **cumulative distribution function (CDF)**, which adds up the probabilities of all possible values of $X$ that is at or below a specific threshold:

$$
F_X(x) = P(X \leq x) = \sum_{x_i \leq x} P(X = x_i)
$$

#### Example

Consider a fair six-sided die. Let $X$ be the outcome of the die roll. The sample space will then be $\Omega = \{1, 2, 3, 4, 5, 6\}$. Since $P(X = x)$ is $\frac{1}{6}$ for all $x \in \Omega$, the PMF for $X$ is

$$
p_X(x) = \frac{1}{6}
$$

and the CDF of $X$ is

$$
F_X(x) = \sum_{x_i \leq x} \frac{1}{6}
$$

### Continuous random variables

The ideas related to discrete random variables are pretty similar to its continuous counterpart. Except, a PMF wouldn't work for continuous random variables because the probability that the variable equals a single value is zero, which doesn't make sense in practice.

#### Probability density function

Instead of a PMF, continuous random variables have a **probability density function (PDF)**. This function describes how the probability is spread out rather than at singular values.

Let's define its PDF $f_X(x)$ as the probability at a *very tiny* interval around $x$ divided by the length of the interval (to get its height):

$$
f_X(x) = \lim_{\delta \to 0} \frac{P(x \leq X \leq x + \delta)}{\delta}
$$

Since the CDF $F_X(x) = P(X \leq x)$, $P(x \leq X \leq X + \delta) = F(x + \delta) - F(x)$, and we can perform a substitution and see that:

$$
f_X(x) = \lim_{\delta \to 0} \frac{F(x + \delta) - F(x)}{\delta} = \frac{dF_X(x)}{dx}
$$

given that $F_X(x)$ is differentiable at $x$. Thus, the PDF is the derivative of the CDF.

The PDF must also satisfy non-negativity and the conditions for total probability:

* $f_X(x) \geq 0$
* $\int_{-\infty}^{\infty} f_X(x) \,dx = 1$

Because $f_X(x) = \frac{dF_X(x)}{dx}$, we can get the CDF by integrating:

$$
F_X(x) = P(X \leq x) = \int_{-\infty}^{x} f_X(t) \,dt
$$

#### Example

Let $X$ represent the lifetime, in hours, of a light bulb made with a novel material. After lots of experimenting, scientists found the the PDF of $X$ to be close to

$$
f_X(x) = \begin{cases}
\frac{1}{2}e^{-\frac{x}{2}} & x \geq 0 \\[2mm]
0                           & x < 0
\end{cases}
$$

The scientists want to find the probability that a randomly-selected light bulb lasts $5$ hours or more. We can find the probability like this:

$$
\begin{aligned}
P(X \geq 5) &= \int_{5}^{\infty} \frac{1}{2}e^{-\frac{x}{2}} \,dx \\[2mm]
            &= \lim_{t \to \infty} \left(-e^{-\frac{t}{2}} - \left(-e^{-\frac{5}{2}}\right)\right) \\[2mm]
            &= e^{-\frac{5}{2}} \\[2mm]
            &\approx 0.08
\end{aligned}
$$

Therefore, a randomly-selected light bulb has about an $8.21\%$ chance of lasting $5$ or more hours.

*Problem 2.* A biased coin has probability $p$ of landing on heads. The coin is flipped repeatedly until the first head appears. Let $X$ be the number of flips made. what is the PMF of $X$?

<MiniCollapsible>
  <MiniCollapsibleTrigger>Solution</MiniCollapsibleTrigger>
  <MiniCollapsibleContent>
    To solve this, we need to find the probability of each outcome.

    To get heads on the $k$-th flip, the first $k-1$ flips must all be tails. The probability of flipping tails is $1-p$, so the probability of flipping heads $k-1$ times in a row is $(1-p)^{k-1}$. The $k$-th flip must be heads, which has a probability of $p$.

    Multiplying these probabilities gives us the PMF

    $$
    p_X(k) = (1-p)^{k-1}p,
    $$

    where $k \in \{1, 2, 3, \dots\}$.
  </MiniCollapsibleContent>
</MiniCollapsible>

*Problem 3.* Let $X$ be a continuous random variable with the PDF $f_X(x) = \frac{c}{x(x+1)}$, where $x \geq 1$. Which of the following is a possible value for $c$?

<MiniCollapsible>
  <MiniCollapsibleTrigger>Solution</MiniCollapsibleTrigger>
  <MiniCollapsibleContent>
    The problem is asking for a value for $c$ that makes $f_X(x)$ a valid PDF. This means it must satisfy the conditions
    
    $$
    f_X(x) > 0 \; \forall x, \qquad \int_{-\infty}^{\infty} f_X(x) \,dx = 1.
    $$

    Since $x > 1$ always, the denominator is also always positive, so non-negativity is already satisfied. Now, let's find a value for $c$ where the second condition is satisfied by setting up and solving the equation

    $$
    \begin{align*}
    \int_{1}^{\infty} \frac{c}{x(x+1)} \,dx                           &= 1 \\[4mm]
    c \int_{1}^{\infty} \left(\frac{1}{x} - \frac{1}{x+1}\right) \,dx &= 1 \\[4mm]
    \lim_{t \to \infty} c(\ln(x) - \ln(x+1))|_1^t                     &= 1 \\[4mm]
    c \ln(2)                                                          &= 1 \\[4mm]
    c                                                                 &= \frac{1}{\ln(2)}
    \end{align*}
    $$
  </MiniCollapsibleContent>
</MiniCollapsible>

        $$

        F_Y(y) &= P(Y \leq y) \\
               &= P(X^2 \leq y) \\

               &= P(-\sqrt{y} \leq X \leq \sqrt{y})

        Using the definition $P(a \leq X \leq b) = F_X(b) - F_X(a)$, we can say that $F_Y(y) = F_X(\sqrt{y}) - F_X(-\sqrt{y})$. Recall that $X$ is uniform on $[-1, 1]$, so $F_X(x) = \frac{x - a}{b - a} = \frac{x + 1}{2}$. With this, we see that $F_Y(y) = \sqrt{y}$.

        Finally, differentiating $F_Y(y)$ gives us $f_Y(y) = \frac{1}{2\sqrt{y}}$.
