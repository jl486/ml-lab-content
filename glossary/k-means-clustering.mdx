## K-means clustering

An <Link href="/glossary#unsupervised-learning">unsupervised learning</Link> algorithm used to group data into specific number of clusters $k$.

The k-means clustering algorithm works like this:

1. Initialize <Link href="/glossary#centroid">centroids</Link> randomly.
2. Assign data points to the nearest centroid.
3. Update centroids to be the mean of the points in the cluster
4. Repeat steps 2 and 3 until centroids <Link href="/glossary#convergence">converge</Link>.
